# rp2354b
Board for implementation testing based on the RP2354B MCU from Raspberry Pi.  
Designed by Ryan Kim in LVMLabs (lvmlabs.org), with the support of Hack Club Stardance (stardance.hackclub.com).
<img width="2362" height="1371" alt="image" src="https://github.com/user-attachments/assets/0a7210c2-8155-4648-8a9a-e73b315e3307" />

### Purpose & Motive
I was looking to implement the RP2354X family of MCUs into my projects, however, I had heard that desigining a board and proper schematic may be difficult. So, I have created this testboard to verify the functionality.

## Features
* 48 GPIOs, which includes 8 ADC channels
* On-silicon 2MB flash
* All pins headered out
